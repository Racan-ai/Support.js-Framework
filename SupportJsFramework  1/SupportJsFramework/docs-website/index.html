Support.js Framework

Comprehensive JavaScript utility framework with React, Next.js, and Angular integrations

npm install support-js-framework react react-dom next @angular/core @angular/common rxjs

Installation Options

All Frameworks

npm install support-js-framework react react-dom next @angular/core @angular/common rxjs

React Only

npm install support-js-framework react react-dom

Next.js Only

npm install support-js-framework react react-dom next

Angular Only

npm install support-js-framework @angular/core @angular/common rxjs

Quick Start

import { formatDate, formatCurrency, isEmail } from 'support-js-framework/core'; import { useLocalStorage, useDebounce } from 'support-js-framework/react'; // Core utilities work everywhere const today = formatDate(new Date(), 'YYYY-MM-DD'); const price = formatCurrency(1234.56, { currency: 'USD' }); const valid = isEmail('user@example.com'); // React hooks for enhanced functionality function App() { const [value, setValue] = useLocalStorage('key', 'default'); const debouncedValue = useDebounce(value, 300); return <div>{debouncedValue}</div>; }
